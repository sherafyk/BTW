# AI News Pipeline

This project demonstrates a minimal implementation of an AI assisted news publishing pipeline.

## Requirements
- Docker

## Running Locally

```bash
docker compose up --build
```

Open `http://localhost:8000/` in your browser and submit a news article URL.
